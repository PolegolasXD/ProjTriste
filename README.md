## Como rodar o projeto ?
> Para rodar o projeto basta rodar no terminal o arquivo `run.sh`

```sh
./run.sh
```

### Erro permission denied
> Erro de permissão, basta dar permissão de leitura para o script

```sh
sudo chmod +x run.sh
```
